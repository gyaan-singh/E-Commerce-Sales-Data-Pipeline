E-Commerce Sales Data Pipeline

 Overview
End-to-end ETL pipeline that processes raw e-commerce sales data and loads it into a MySQL data warehouse for business analytics.

Tech Stack
- Python (Pandas, ETL)
- MySQL (Data Warehouse)
- SQL (Analytics)

Features
- Data extraction from CSV
- Data cleaning & transformation
- Derived metrics (total revenue)
- SQL-based analytics
- Business insights generation

 Business Insights
- Total revenue calculation
- Best-selling products
- Category-wise performance
- City-wise sales distribution
